How it works. The tool tries four packing strategies and shows the densest result:
  1. AABB column — stack identical bounding boxes along the part's shortest axis.
  2. AABB 3D grid — pick the (nx, ny, nz) layout that minimizes the longest outer side.
  3. Mesh nesting (translation) — BVH-accelerated triangle-mesh collision, binary-searches the smallest non-overlap offset on each axis. Catches cups, U-channels, and trays that slot into themselves.
  4. Mesh nesting (alternating flip) — every second part rotated 180° around one axis. Mating opposite orientations works well for trapezoidal sheet-metal profiles, Z-sections, and other parts that don't nest by pure translation.
Some parts (closed brackets, fully solid blocks) don't nest at all — the tool falls back to AABB and reports that honestly.

Not yet modeled: rotational nesting (continuous rotation rather than 180° flip), multi-axis offsets, mesh-vs-mesh interlocking. These could squeeze a bit more density on certain parts.
